# Web Calculator

A simple and stylish web calculator built with HTML, CSS, and JavaScript.  
Demonstrates basic arithmetic operations and interactive UI elements.

## Technologies
- HTML5
- CSS3
- JavaScript

## Features
- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal input support
- Clear (AC) and delete (DE) buttons
- Responsive and visually appealing design
- Hover effects and button animations

## How to run
1. Open `index.html` in your browser.
2. Use the calculator by clicking the buttons.
